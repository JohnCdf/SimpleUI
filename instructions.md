# Instructions

## Head

Here's how the head of the document could look like

```html

<!-- Javascript and CSS includes -->  
<link rel="stylesheet" type="text/css" href="lib/simpleUI.css" />
<script type="text/javascript" src="lib/jquery-1.7.1.min.js"></script>
<script type="text/javascript" src="lib/simpleUI.js"></script>

<!-- My theme -->
<link rel="stylesheet" type="text/css" href="lib/myTheme.css" />		
		


<!-- Other Javascript files you whish to include -->
<script src="otherLibraries/shCore.js" type="text/javascript"></script>		
<script src="otherLibraries/shBrushXml.js" type="text/javascript"></script>
<link rel="stylesheet" type="text/css" href="otherLibraries/shCoreDefault.css" />
<script type="text/javascript" src="otherLibraries/myOtherScript.js"></script>

		
<!-- Vieport and appable -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no" />
<meta name="apple-mobile-web-app-capable" content="yes" />

```
